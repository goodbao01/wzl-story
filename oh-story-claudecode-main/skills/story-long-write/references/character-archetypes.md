# 角色类型谱系参考

写作时确保每个角色在读者心中有**明确辨识度**。一个角色让读者记住，靠的不是外貌描写，而是**他说话的方式、他做决定的方式、他在压力下的反应**。

---

## 一、核心角色类型

每个角色从下面选 1 个主类型 + 可选 1 个副类型。主类型决定核心行为模式，副类型提供反差。

### A类：热血莽撞型

| 维度 | 特征 |
|------|------|
| **核心驱动** | 情绪 > 理智 | 冲动先于思考 | "忍不了" |
| **说话节奏** | 短句为主，喜欢用感叹号 | 经常话没说完就行动 | "少废话，干他！" |
| **情绪表达** | 外放，喜怒都在脸上 |
| **压力反应** | 迎上去，不后退 |
| **思维模式** | 线性直白，不绕弯 | 容易被激将 |
| **弱点** | 容易中计、冲动坏事、不懂藏锋 |
| **标志句式** | "让我来！""凭什么！""你再说一遍！" 反问式爆发 |
| **常用语气** | 急促、高亢、不耐烦 |

**适配角色**：热血主角、冲动型伙伴、少年、武痴、憨直型角色
**写作提示**：对话时保持短句节奏，长句会崩人设。行动永远先于思考，不要让他"想了想再说"。

---

### B类：冷静谋略型

| 维度 | 特征 |
|------|------|
| **核心驱动** | 理智 > 情绪 | 计算先于行动 | "让我想想" |
| **说话节奏** | 长句为主，喜欢设问和反问 | 经常沉默后再回答 |
| **情绪表达** | 内敛，喜怒不行于色 |
| **压力反应** | 后退一步，观察评估 |
| **思维模式** | 推演、比较、找最优解 |
| **弱点** | 犹豫不决、冷血无情、过度算计反噬 |
| **标志句式** | "你有没有想过…""如果…那么…""不一定。" 设问式推进 |
| **常用语气** | 平缓、克制、略带距离感 |

**适配角色**：军师型、幕后黑手、操控师、冷静主角
**写作提示**：不要让他一次性说完所有想法。用"沉默→简短回应→事后证明他早就想到了"制造谋略感。对话里多用设问诱导对方说出他想要的。

---

### C类：吐槽役 / 搞笑担当

| 维度 | 特征 |
|------|------|
| **核心驱动** | 解构 > 参与 | 吐槽先于正经 | "不是，你认真的？" |
| **说话节奏** | 喜欢接话、拆台、补刀 | 节奏感强，擅长三句话造一个笑点 |
| **情绪表达** | 外放但带有距离感（吐槽本质是保护色） |
| **压力反应** | 用玩笑缓解紧张 / 沉默（压力太大时不吐槽） |
| **思维模式** | 类比思维：能找到两个不相关事物的荒诞联系 |
| **弱点** | 关键时刻不够正经、被人当小丑、插科打诨耽误正事 |
| **标志句式** | "好家伙…""你管这叫…？""我谢谢你啊。" 场景解构式吐槽 |
| **常用语气** | 夸张、反差、一本正经地说胡话 |

**适配角色**：主角好友、旁白吐槽、气氛组、解压器
**写作提示**：吐槽频率要看场合。高压场景里让吐槽役闭嘴，反而更显张力。吐槽的节奏是**铺→转折→爆**，不要直白说"好搞笑"而是用逻辑反差制造笑点。

---

### D类：冰山面瘫型

| 维度 | 特征 |
|------|------|
| **核心驱动** | 规则 > 人情 | 行动 > 语言 | "……" |
| **说话节奏** | 字数极少 | 能用点头回答就不说话 | 惜字如金 |
| **情绪表达** | 几乎没有表情 | 用微动作/沉默表达不悦 |
| **压力反应** | 面不改色地执行方案 |
| **思维模式** | 极端实用主义 | 结果导向 |
| **弱点** | 不擅长沟通、被误认为冷漠、缺少人情味 |
| **标志句式** | "嗯。""走。""杀了。" 动词+宾语式短句 |
| **常用语气** | 没有任何语气词 | 陈述句为主 |

**适配角色**：冷面杀手、高冷女神、寡言强者、护卫型
**写作提示**：最容易被写成"面瘫但内心戏多"——千万别。冰山角色的内心和脸一样平，或者至少读者不能直接看到他的内心。他的情绪通过**行为结果**来体现：他默默替主角挡了一刀，比说一百句"我在乎你"更有效。

---

### E类：社交型 / 话痨型

| 维度 | 特征 |
|------|------|
| **核心驱动** | 连接 > 独处 | 说话就是思考 |
| **说话节奏** | 话多、语速快、经常跑题 | 一个人能说三个人的对话量 |
| **情绪表达** | 极度外放，夸张化表达 |
| **压力反应** | 不停说话来缓解焦虑 |
| **思维模式** | 发散式联想 | 跳跃性思维 |
| **弱点** | 话多误事、守不住秘密、被人嫌烦 |
| **标志句式** | "我跟你说啊…""对了还有…""你猜怎么着？" 串珠式叙述 |
| **常用语气** | 热情、急迫、自问自答 |

**适配角色**：话痨队友、情报通、百事通、气氛组
**写作提示**：话痨的对话可以适当插话、打断、跑题，这是真人的说话方式。但注意：话痨不等于废话多，他的话里要有信息量——用话多的方式给情报。

---

### F类：傲娇型 / 口是心非型

| 维度 | 特征 |
|------|------|
| **核心驱动** | 自尊 > 真实感受 | 嘴上说的 ≠ 心里想的 |
| **说话节奏** | 先否定再肯定 | 用反问掩饰在意 |
| **情绪表达** | 真实情绪藏在攻击性语言后面 |
| **压力反应** | 防御性攻击："关你什么事" |
| **思维模式** | 高自尊驱动，死要面子 |
| **弱点** | 错过机会、被人误解、嘴硬害死人 |
| **标志句式** | "谁稀罕！""我才不是…""随便你。" 否定前置 + 身体诚实 |
| **常用语气** | 不耐烦但实际在意 | 攻击性防御 |

**适配角色**：傲娇女主、嘴硬搭档、死对头
**写作提示**：傲娇的精髓是"嘴和身体不一致"。嘴上说"我才不去"，脚已经往那个方向走了。用身体出卖语言。

---

### G类：病娇 / 偏执型

| 维度 | 特征 |
|------|------|
| **核心驱动** | 执念 > 一切 | "你是我的" |
| **说话节奏** | 平时正常，提到执念时节奏突变 |
| **情绪表达** | 平静表象下的极度不稳定 |
| **压力反应** | 触发执念→极端行为 |
| **思维模式** | 单点聚焦，其他都不重要 |
| **弱点** | 不可预测、自毁倾向、把喜欢的人推开 |
| **标志句式** | "你只能是我的。""没有你我活不下去。""谁也别想把你带走。" 重复式执念 |
| **常用语气** | 温柔时极温柔，疯狂时极疯狂 | 两极切换 |

**适配角色**：偏执反派、疯批美人、极端守护者
**写作提示**：病娇的恐怖感来自**切换**——前一句还在笑着说情话，后一句眼神就变了。不要让病娇一直疯，平静中的爆裂才是精髓。

---

### H类：忠犬 / 守护型

| 维度 | 特征 |
|------|------|
| **核心驱动** | 忠诚 > 自我 | "我会保护你" |
| **说话节奏** | 直接、朴素、不绕弯 |
| **情绪表达** | 直率但不夸张 |
| **压力反应** | 挡在前面，不计代价 |
| **思维模式** | 二元：自己人/外人 | 对错分明 |
| **弱点** | 愚忠、没主见、容易被利用 |
| **标志句式** | "交给我。""有我在。""你放心。" 承诺式短句 |
| **常用语气** | 坚定、简单、可靠 |

**适配角色**：忠实护卫、青梅竹马、可靠大哥、老实人
**写作提示**：忠犬的魅力在于**无声的牺牲**——他不是用嘴说"我在乎你"，而是用行动证明。他的对话越简单越有力。

---

### I类：高深莫测型（Boss 位）

| 维度 | 特征 |
|------|------|
| **核心驱动** | 目的 > 情绪 | 每一步都有深意 |
| **说话节奏** | 慢、少、每一句都有分量 |
| **情绪表达** | 从不出底牌，永远从容 |
| **压力反应** | 越紧张越从容（反常识） |
| **思维模式** | 多步推演，想到读者没想到的层面 |
| **弱点** | 过度自信、低估对手的非常规操作、不愿意亲自动手 |
| **标志句式** | "你终于到了这一步。""我等这一天很久了。" 双关语、多义句 |
| **常用语气** | 从容、温和但让人毛骨悚然 |

**适配角色**：幕后Boss、老谋深算的长者、高阶存在
**写作提示**：高深莫测 ≠ 谜语人。他说的话要有两层意思——表面一层给角色听，深层一层给读者品。每句话都要有弦外之音。

## 二、角色关系张力表

不同角色类型配对会产生特定的**化学反应**：

| 组合 | 张力类型 | 示例场景 |
|------|----------|----------|
| A（热血） + B（谋略） | 互补冲突 | A要冲，B要等→"你要忍到什么时候！""等到能赢的时候。" |
| A（热血） + D（冰山） | 冷热反差 | A疯狂输出情绪，D只回"嗯"→A被逼疯 |
| C（吐槽） + D（冰山） | 单口相声 | C疯狂吐槽，D完全没反应→C更用力吐槽→观众笑 |
| C（吐槽） + F（傲娇） | 互相伤害 | C戳破F的真实想法→F暴怒否认→读者：哈哈哈哈 |
| E（话痨） + D（冰山） | 棉花碰铁板 | E说300字，D回3个字→E说更多→戏剧效果 |
| F（傲娇） + H（忠犬） | 口嫌体正直 | F说"我才不管他"，H真的走了→F着急 |
| G（病娇） + B（谋略） | 威胁对算计 | G不按常理出牌，B算不到→B被压制 |
| I（高深） + A（热血） | 降维打击 | I轻轻一句话，A全力一拳打空 |
| H（忠犬） + I（高深） | 忠诚考验 | I设局测试H的忠诚→H的选择定义故事走向 |
| B（谋略） + F（傲娇） | 看穿不说穿 | B看出F的心意但不点破→F更着急 |

> **原则**：每个互动场景至少存在一个张力关系。没有张力关系的对手戏是白开水。

## 三、角色成长线参考

| 类型 | 常见成长方向 | 反面堕落方向 |
|------|-------------|-------------|
| A 热血 | 学会克制和思考 | 变得鲁莽害死同伴 |
| B 谋略 | 学会信任和冒险 | 算计过度众叛亲离 |
| C 吐槽 | 在关键时刻正经起来 | 永远停留在搞笑角色 |
| D 冰山 | 学会表达和依赖 | 彻底封闭自我 |
| E 话痨 | 学会倾听和沉默 | 话多误事 |
| F 傲娇 | 学会坦诚表达 | 嘴硬到底失去一切 |
| G 病娇 | 学会放手和信任 | 执念成魔 |
| H 忠犬 | 学会有自己的判断 | 愚忠走向毁灭 |
| I 高深 | 学会亲自动手 | 傲慢招致败局 |

## 四、避坑指南

| 常见问题 | 原因 | 解决方法 |
|----------|------|----------|
| 角色说话都一个味 | 没做角色分类 | 每个人物建档时写下 3 个对话特征 |
| A类角色不冲动 | 作者怕角色犯错 | 让角色犯错，读者爱的不是完美而是成长 |
| 冰山角色内心戏太多 | 误解了"面冷心热" | 冰山角色的内心也要冷，热在行动上 |
| 吐槽役不分场合吐槽 | 为了搞笑牺牲沉浸感 | 高压场景让吐槽役闭嘴，张力反而更强 |
| 所有角色都用"说" | 缺乏对话标记变化 | 根据性格：急性子用"吼/嚷"，冷淡的用"淡淡道"或无标记 |
| 角色之间没有化学反应 | 没考虑张力配对 | 每个场景至少带一个有张力关系的对手戏 |
