# 起点中文网平台适配指南

起点中文网（付费订阅模式）的读者期望和写作禁忌。

## 平台特征

| 维度 | 特征 |
|------|------|
| 读者画像 | 资深网文读者、追求深度和体系的付费用户、年龄 18-35 |
| 付费模式 | 千字 3-5 分钱订阅，依赖章节订阅量和追订率 |
| 黄金章节 | 免费期前 20 章决定收藏，上架后前 3 章决定追订 |
| 平台算法 | 追订率 > 收藏 > 月票 > 推荐票 |

## 读者期望

| 期望 | 说明 |
|------|------|
| 世界观完整 | 设定合理、力量体系清晰、不崩 |
| 角色有深度 | 不是工具人，有成长线和动机 |
| 爽点有铺垫 | 不白爽，铺垫→释放节奏清晰 |
| 长篇抗造 | 百万字级别的设定和剧情不崩 |
| 章节有信息量 | 每章要有推进，不只是水字数 |

## 写作禁忌

| 禁忌 | 说明 |
|------|------|
| 开局无敌 | 起点读者不喜欢开局满级，喜欢成长过程 |
| 战力随意崩 | 等级体系说改就改、前期 boss 变后期杂兵 |
| 长时间无主线 | 超过 20 章游离主线 → 掉追订 |
| 弱智反派 | 反派太蠢 → 爽感降低 |
| 女主变花瓶 | 感情线处理不好 → 女读者流失 |

## 字数建议

| 节奏 | 建议字数 | 说明 |
|------|----------|------|
| 正常章 | 2500-3500 字 | 订阅性价比合理 |
| 高潮章 | 3000-4000 字 | 读者能接受付费看高潮 |
| 过渡章 | 2000-2500 字 | 保证有信息量 |

## 写入 prompt 的平台指令

```markdown
### 起点平台适配
- 世界观设定要合理自洽，力量体系清晰
- 角色有成长线和动机，不做工具人
- 爽点要有充分铺垫再释放，不白爽
- 反派智商在线，不为了爽而强行降智
- 主线推进要持续，不要超过 10 章游离主线
- 感情线处理要细腻
```
