# GPT 模型适配指南

GPT 系列模型（GPT-4o / GPT-4 / GPT-4-turbo）在网文写作中的常见弱点及纠偏指令。

## 常见弱点

### 1. 过度解释剧情
**表现**：写完一段情节后，忍不住解释"这意味着……""原来……""这说明……"
**纠偏指令**：禁止在叙事中使用"原来""原来如此""这意味着""总而言之""显然""不得不说"等总结词。
**严重程度**：高

### 2. 总结主题
**表现**：章末或段尾出现"这就是……的真谛""这就是成长"等价值观输出
**纠偏指令**：禁止章末总结、禁止替读者提炼主题。让事件自己说话。
**严重程度**：高

### 3. 过度正确
**表现**：角色不敢做道德上有争议的事，好人太好坏人太坏，缺灰色地带
**纠偏指令**：允许角色犯错、冲动、嘴硬、偏执。不要求角色"正确"。
**严重程度**：中

### 4. 人生感悟泛滥
**表现**：角色动不动就来一段人生感悟，像哲学家附体
**纠偏指令**：减少角色内心独白中的大道理。让行动表达态度。
**严重程度**：中

### 5. 对称句式和排比滥用
**表现**：喜欢用"不是……而是……""既……又……""虽然……但是……"等对称结构
**纠偏指令**：减少对称句式，增加短促口语化表达。
**严重程度**：中

## 纠偏开关

```
model_adapter: gpt
fixes:
  - over_explain: true    # 开启：禁止解释剧情/总结/说教
  - perfect_character: true  # 开启：允许角色犯错/不完美
  - philosophy: true      # 开启：减少人生感悟
  - symmetry: true        # 开启：减少对称句式
```

## 写入 narrative-writer prompt 的指令

```markdown
### GPT 适配指令
- 禁止解释剧情（不用"这意味着""原来""显然""不得不说"）
- 禁止章末总结、主题提炼
- 允许角色做道德上可质疑的事
- 减少"不是……而是……"等对称句式
- 角色用行动表达态度，不要大段内心独白讲道理
```
