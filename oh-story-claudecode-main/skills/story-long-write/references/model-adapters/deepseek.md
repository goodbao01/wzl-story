# DeepSeek 模型适配指南

DeepSeek 系列模型在网文写作中的常见弱点及纠偏指令。

## 常见弱点

### 1. 人设漂移
**表现**：写着写着角色的性格、语气、行为模式发生偏移，尤其长篇连载时明显
**纠偏指令**：写作前加载角色设定文件，逐条核对角色性格关键词。对话前问"这个角色会这么说吗？"
**严重程度**：高

### 2. 爽点模板化
**表现**：打脸/装逼/震惊的套路高度重复，"全场震惊""倒吸一口凉气"频繁出现
**纠偏指令**：检查最近 10 章重复的爽点模式。第 1 次用"全场震惊"可以，第 5 次必须换写法。
**严重程度**：高

### 3. 桥段重复
**表现**：同类型的冲突/误会/反转在书中反复出现
**纠偏指令**：建立本章桥段清单，与最近 20 章对比，发现重复立即替换。
**严重程度**：高

### 4. 战力崩坏
**表现**：等级体系在长篇中逐渐失控，前期 boss 变后期杂兵不做合理过渡
**纠偏指令**：力量体系升级必须有铺垫、有瓶颈、有代价。不凭空升级。
**严重程度**：高

### 5. 陈词滥调库
**表现**："倒吸一口凉气""目瞪口呆""难以置信""瞠目结舌"高频出现
**纠偏指令**：用脚本扫描这些词，发现即替换为具体描写。
**严重程度**：中

## 纠偏开关

```
model_adapter: deepseek
fixes:
  - character_drift: true   # 开启：写作前加载角色设定核对
  - template_shuang: true   # 开启：检查爽点模式多样化
  - plot_repeat: true       # 开启：对比最近 20 章桥段
  - power_level: true       # 开启：力量体系一致性检查
  - cliche_words: true      # 开启：陈词滥调检测替换
```

## 写入 narrative-writer prompt 的指令

```markdown
### DeepSeek 适配指令
- 逐条核对角色设定：当前对话/行为是否符合该角色的性格关键词
- 避免"全场震惊""倒吸一口凉气""目瞪口呆""难以置信"等模板描写
- 力量体系升级需要有铺垫和代价
- 检查与最近 10 章是否存在同类型桥段重复
```
